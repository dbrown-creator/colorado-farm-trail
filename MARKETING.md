## Launch checklist

1. ✅ Map set to **Anyone with the link → Viewer**.
2. ⏳ Site live on GitHub Pages (see [`ROADMAP.md`](ROADMAP.md)).
3. ⏳ GA4 confirmed in **Realtime**.
4. ☐ Create per-channel tracked links (UTM + Bitly).
5. ☐ Download QR for the flyer link.
6. ☐ Paste links into the posts above and publish.
7. ☐ Check the Bitly + GA4 dashboards weekly; double down on the channel that wins.
