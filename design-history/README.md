# Design history

Visual snapshots of [coloradofarmtrail.com](https://coloradofarmtrail.com) around
notable design changes, so we can see how the site has evolved over time.

## Convention

- **One folder per milestone**, named `YYYY-MM-DD_<short-label>/` (the date the
  snapshot was captured).
- Inside, name each capture by **page + optional viewport + state**, e.g.
  `homepage.png`, `homepage_mobile.png`, `homepage_explore.png`, `map_dark.png`.
- Prefer **PNG** — it renders inline on GitHub and in PRs. PDF is acceptable if
  that's what the capture tool produced (it just won't preview inline).
- Capture the **"before"** just prior to merging a redesign, and an **"after"**
  once it lands, so each change has a matched pair.
